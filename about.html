About RecipeFinder

What We Do

RecipeFinder is a web application that helps you discover new recipes from around the world. Powered by TheMealDB API, we provide access to thousands of recipes across various cuisines and categories.

Features

  • Search for recipes by name or ingredient
  • Filter by category
  • Save your favorite recipes
  • View detailed recipe information
  • Your saved recipes persist across sessions

Technologies Used

  • HTML5 with semantic elements
  • CSS3 with Flexbox and Grid layouts
  • JavaScript ES6+ modules
  • LocalStorage for data persistence
  • RESTful API integration
